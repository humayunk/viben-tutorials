What is OpenClaw?

Think of OpenClaw as a brain for your chatbots. It connects to messaging platforms (WhatsApp, Telegram, Discord) and lets you build AI-powered agents that can:

  • Respond to messages with any AI model (Claude, GPT, Llama)
  • Connect to APIs, databases, and external tools
  • Run automations on a schedule
  • Handle multi-step conversations with memory

It's open-source, self-hosted, and designed for people who want to build useful bots without writing much code.

Step 2 of 5
Install the Prerequisites
Your vibe
You need two things before installing OpenClaw:

1. Docker Desktop
Docker is your safety net. Think of it like a sandbox at a playground — your AI agent can build whatever it wants inside the sandbox, but it can't dig into your yard, your house, or your neighbor's property.

Download from docker.com. Once installed, make sure the whale icon appears in your menu bar.

2. Node.js v22.12+
OpenClaw requires Node.js version 22.12 or higher. Not v20, not v18 — specifically v22.12+.

Download from nodejs.org (pick the LTS version).

Heads up: If you already have an older Node version, you'll need to upgrade. If you use nvm: nvm install 22 && nvm use 22

Step 3 of 5
Install & Run the Wizard
Your vibe
One command to install:

npm install -g openclaw@latest
Then run the setup wizard:

openclaw onboard --install-daemon
You'll see a scary security warning. That's actually a good sign — the developers are being transparent. The risks it describes are real, and that's exactly why we set up Docker. Say Yes to continue.
Wizard settings cheat sheet:

Gateway type → Local
Workspace → Default (just press Enter)
AI Provider → Google Gemini Flash (free) or Anthropic Claude (paid, best quality)
Gateway port → Default (18789)
Gateway bind → Loopback (most secure)
Tailscale → Off
Chat channels → Set up WhatsApp if offered
Skills → Skip
Hooks → Enable boot.md
Runtime → Node

API key ≠ subscription. Even if you pay for Claude Pro ($20/mo), that doesn't give you API credits. The API is billed separately at console.anthropic.com. When in doubt, start with Google's free tier.

Step 4 of 5
Meet Your Agent & Secure It
Your vibe
Launch OpenClaw by typing openclaw in your terminal. Your agent wakes up and introduces itself.

Give it a name, tell it about yourself. The more context you provide, the better it works. It creates a SOUL.md file — your agent's personality config.

Your agent is defined by markdown files:

~/.openclaw/
  agent.md ← Core instructions
  identity.md ← Name, personality
  memory.md ← What it remembers
  soul.md ← Agent identity
  tools.md ← Tool interactions
  skills/ ← Installed skills
Now lock it down:

In the OpenClaw chat, type: openclaw security audit deep

Your agent scans your setup, shows you what it finds, and proposes fixes with your approval. Then enable the Docker sandbox:

Enable Docker sandbox for the default agent on agent scope
Type this in the OpenClaw chat, not your terminal
You're now protected. Your agent can't access your personal files, desktop, or anything outside its container.

Step 5 of 5
Connect WhatsApp & Send Your First Message
Your vibe
If you set up WhatsApp during the wizard, you're already connected — skip to sending a message below.

Otherwise, run these in your terminal (not the OpenClaw chat):

openclaw plugins enable whatsapp
openclaw gateway restart
openclaw channels login
A QR code will appear. Scan it with your phone (WhatsApp → Settings → Linked Devices → Link a Device).

Never add your bot to a group chat. Anyone in the group could send prompts that execute on your machine. Keep it 1-on-1 only.
Send your first message:

Open WhatsApp → tap + → search for your own phone number → send:

What did we set up today? Give me a summary.
Wait a few seconds. Your agent — running on your computer — reads the message, thinks, and responds with a full summary. From your phone. To your computer. And back.
